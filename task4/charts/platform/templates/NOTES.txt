Platform components managed by this chart:

- Gateway API CRDs via the local `gateway-api-crds` dependency
- cert-manager via Helm
- CloudNativePG operator via Helm
- NGINX Gateway Fabric via Helm

Verify installation with:

  kubectl get crd gateways.gateway.networking.k8s.io
  kubectl get deploy -n {{ .Release.Namespace }}
  kubectl get deploy -n nginx-gateway
  kubectl get deploy -n cnpg-system
  kubectl get deploy -n cert-manager
